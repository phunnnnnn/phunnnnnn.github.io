# Phunn — Portfolio

Personal portfolio site for **Phunn (Pannatat Chaikaew)** — developer & creator of AutoBlox.

- **Live site:** https://phunnnnnn.github.io
- **Source:** served from the `docs/` folder (static HTML/CSS/JS, no build step)

## Features
- 🌐 TH / EN language toggle
- 🌙 Dark / Light theme toggle
- 📱 Responsive layout with mobile menu

## Editing
All content lives in `docs/`:
- `index.html` — page structure
- `styles.css` — styling & theme tokens
- `script.js` — theme toggle, language toggle, and the `translations` object (edit both `en` and `th` when changing text)
- `assets/phunn.png` — profile icon
